# User Report
A report to track which users are logged in to which machines.

Problem statement: Process a list of Event objects using their attributes to generate a report that lists all the users currently logged in to the machines.

Research: Input must be sorted by date. Use a function to chronologically order the elements of the list.

Planning: Use a dictionary of sets to store the current users along with the machines they are using. Once the events are processed, print a report.

Writing the script: Use a function to create the dictionary, a second one to sort it, and third one to print it.

Putting it all together: Store and debug the code in a Jupyter Notebook.
