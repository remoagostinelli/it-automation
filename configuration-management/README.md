# Configuration Management and the Cloud
Apply automation to manage fleets of computers. Automate the process for deploying new computers, keeping those machines updated, managing large-scale changes, and a lot more. Manage both physical and virtual machines running in the Cloud.

Start by looking into an automation technique called configuration management, which allows to manage the configuration of our computers at scale.
Puppet is the current industry standard for configuration management. Look at some simple examples, and then see how we can apply the same concepts to more complex cases.

Later on, expand your automation skills by understanding how to use the Cloud to help scale your infrastructure. Check out some best practices for handling hundreds of virtual machines running in the Cloud and troubleshooting them when things don't go according to plan.
