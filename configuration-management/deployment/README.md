# Deployment Using Puppet
To manage the computers in your company's fleet, including a bunch of different machines with different operating systems, you can use Puppet to automate the configurations on these machines. Part of the setup is already done, but there are more rules that need to be added, and more operating systems to consider.

Prerequisites:

- How to create Puppet classes and rules
- How Puppet interacts with different OSs
- How to use the DSL to create complex rules
