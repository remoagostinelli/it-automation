# Create VM Template and Automate Deployment
Deploy eight virtual machines (VMs) as web servers. Each of them should have the same configuration. You'll create a VM, set up an auto-enabled service, and make it a template. Then, you'll use the template to create seven more VMs.
- Create a VM using GCP web UI and make a template out of it
- Use a command-line interface to interact with VMs
- Configure an auto-enabled service
- Use gcloud to deploy VMs with a template
